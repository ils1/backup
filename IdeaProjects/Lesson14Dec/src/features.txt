Телефонная книга
------------------------------------------------
+ Создание новой пустой телефонной книги
- Получение числа контактов
- Добавление нового контакта с именем и номером телефона
- При добавлении контакта с уже существующим именем номер телефона добавляется к сущ-му контакту.
- При добавлении контакта с уже существующим номером создаётся новый контакт
- Добавление номера телефона существующего контакта
- Удаление номера телефона контакта
- Удаление самого контакта
- Поиск номеров контакта по имени
- Поиск контакта по фрагменту имени
